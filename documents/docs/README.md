## 简介
王者营地自动任务工具

## 功能
* 签到
* 完成每日任务
* 领取任务奖励
* 支持多账号
* 微信推送
* 【温馨提示】华为云函数将于2023年9月1日结束免费服务
* 项目已迁移至LeanCloud，免费且自动化运行
